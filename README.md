## Stack

- React
- Redux
- React-Router
- Styled-Components

## Development

1. Install dependencies

   ```sh
   yarn install
   ```

2. Create an `.env` file from the `.env.example` file

3. Start the app

   ```sh
   yarn start
   ```

## Tests

```
yarn test
```
